# 方法

在 Typora 中导出侧栏大纲，可以按照以下步骤操作：

1. **打开文件**：首先在 Typora 中打开你需要导出大纲的 Markdown 文件。
2. **显示侧栏大纲**：确保侧栏大纲已经显示。如果没有显示，可以通过以下方式打开：
	- 在菜单栏中选择 `视图` > `大纲`。
	- 或者使用快捷键 `Ctrl + Shift + 1`（Windows）或 `Cmd + Shift + 1`（Mac）。
3. **导出大纲**：
	- Typora 本身没有直接导出侧栏大纲的功能。不过，你可以通过复制大纲内容并手动创建一个新文件来实现这一目的。
	- 在侧栏大纲中，手动选择并复制所有大纲内容。
	- 打开一个新的 Typora 窗口或其他文本编辑器，将复制的大纲内容粘贴进去。
	- 保存该文件为一个新的 Markdown 文件。
4. **另存为其他格式**（可选）：
	- 如果你需要将大纲保存为其他格式（如 PDF、HTML、Word 等），可以在保存后的 Markdown 文件中选择 `文件` > `导出`，选择你需要的格式进行导出。

通过上述步骤，你可以在 Typora 中导出侧栏大纲的内容，并保存为你需要的文件格式。
